A simple Elementor widget that allows you to create job postings and automatically generates JobPosting structured data for improved SEO visibility.

== Description ==

**Job Posting Widget for Elementor** is an easy-to-use addon for the Elementor page builder. It allows you to create well-structured job listings using a dedicated widget and automatically generates **JobPosting schema markup** based on the information you enter.

Whether you're an employer, recruiter, or job board operator, this plugin helps you quickly publish job listings with clean design and correct structured data — helping improve visibility in Google search and potentially eligibility for Google for Jobs (results are not guaranteed).

### What it does:

- Adds a "Job Posting" widget to Elementor  
- Lets you enter job title, description, employment type, salary, location, and more  
- Generates valid JobPosting schema automatically  
- Displays a job listing block you can style freely using Elementor  
- Works with any theme that supports Elementor

This plugin focuses on making job publishing easy, clean, and SEO-friendly without requiring external services or complicated configuration.

== Features ==

- Native Elementor widget for creating job listings  
- Automatic generation of JobPosting structured data  
- Customizable layout using Elementor’s design controls  
- Lightweight and optimized for performance  
- Responsive output that looks great on all devices  
- Works with both Elementor Free and Elementor Pro

== Installation ==

1. Install and activate the Elementor plugin (required).  
2. Upload the plugin files to `/wp-content/plugins/job-posting-widget-for-elementor/`  
   or install it via **Plugins → Add New**.  
3. Activate **Job Posting Widget for Elementor**.  
4. Open the Elementor editor on any page.  
5. Search for the **"Job Posting"** widget and drag it onto your layout.  
6. Fill in your job details and style the block as you wish.  
7. Publish the page and test your structured data using Google’s Rich Results Test.

== Frequently Asked Questions ==

= Does this plugin require Elementor Pro? =
No. It works with Elementor Free. Elementor Pro is optional.

= Does the plugin automatically add schema markup? =
Yes. As soon as you fill in the widget fields, the schema is generated automatically.

= Will my job appear in Google for Jobs? =
The plugin creates the correct structured data, but inclusion in Google for Jobs cannot be guaranteed.

= Does it work with any theme? =
Yes. Any theme compatible with Elementor will work.
